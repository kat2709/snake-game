Hungry Caterpillar

Score : 0

Use "ArrowLeft" or "A" to make the snake go left, "ArrowRight" or "D" to make the snake go right, "ArrowUp" or "W" to make the snake go up, and "ArrowDown" or "S" to make the snake go down. The snake grows after eating an apple. You get 100 points for every eating apple. Snake can go through a wall. If the snake hits snake's tail game over! For mobile version use navigation arrows to change snake's direction. Good luck and have fun!
